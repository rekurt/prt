## Что такое prt?

`prt` показывает, какие процессы занимают сетевые порты — в реальном времени, прямо в тер��инале. Интерактивный аналог `lsof -i` / `ss -tlnp` с подсветкой, фильтрацией и деревом процессов.

## Возможности

| Функция | Описание |
|---------|----------|
| **Живая таблица** | Порты, сервисы, протоколы, состояния, PID, процессы, пользователи. Обновление каждые 2с |
| **Отслеживание изменений** | Новые соединения — зелёным, закрытые — красным на 5с |
| **Детектор подозрительных** | Метка `[!]` для non-root на привилегированных портах, скриптов на чувствительных портах |
| **Дерево процессов** | Цепочка родителей (launchd → nginx → worker) |
| **Панель деталей** | Дерево / Сеть / Соединение для выбранного процесса (`1` `2` `3`) |
| **Полноэкранные режимы** | График (`4`), Топология (`5`), Детали процесса (`6`), Namespaces (`7`) |
| **Поиск и фильтр** | По порту, сервису, имени, PID, протоколу, состоянию, пользователю. `!` = подозрительные |
| **Сортировка** | По любой колонке, по возрастанию/убыванию |
| **Завершение процесса** | Выбрать → `K` → подтвердить `y` (SIGTERM) или `f` (SIGKILL) |
| **Блокировка IP** | `b` → блокировка удалённого IP через iptables/pfctl с командой отмены |
| **Strace** | `t` → трассировка syscalls в split-панели |
| **Sudo** | Нажать `s`, ввести пароль — увидеть все процессы системы |
| **Буфер обмена** | Копировать строку (`c`) или PID (`p`) |
| **Контейнеры** | Имя Docker/Podman контейнера (скрывается если нет контейнеров) |
| **Bandwidth** | Системный RX/TX в заголовке |
| **Экспорт** | `prt --export json`, `prt --export csv`, `prt --json` (NDJSON потоком) |
| **Watch режим** | `prt watch 3000 8080` — компактный UP/DOWN монитор |
| **Алерты** | TOML-конфиг с bell/highlight по порту, процессу или числу соединений |
| **Мультиязычность** | Английский, русский, китайский. Автоопределение, переключение `L` |
| **Конфигурация** | `~/.config/prt/config.toml` — кастомные имена портов, правила алертов |

## Использование

```bash
prt                     # запустить TUI
prt --lang ru           # русский интерфейс
prt --export json       # экспорт в JSON
prt --export csv        # экспорт в CSV
prt --json              # NDJSON потоком в stdout
prt watch 80 443 5432   # компактный мониторинг портов
sudo prt                # запуск от root (все процессы)
```

## Горячие клавиши

**Навигация:**

| Клавиша | Действие |
|---------|----------|
| `j`/`k` `↑`/`↓` | Перемещение / скролл |
| `g` / `G` | В начало / конец |
| `/` | Поиск и фильтр (`!` = подозрительные) |
| `Esc` | Назад к таблице / сбросить фильтр |
| `q` | Выход |

**Нижняя панель (режим таблицы):**

| Клавиша | Действие |
|---------|----------|
| `Enter` / `d` | Показать/скрыть панель деталей |
| `1` `2` `3` | Дерево / Сеть / Соединение |
| `←`/`→` `h`/`l` | Переключение вкладок |

**Полноэкранные режимы:**

| Клавиша | Действие |
|---------|----------|
| `4` | График — соединения по процессам |
| `5` | Топология — процесс → порт → удалённый |
| `6` | Детали процесса — инфо, файлы, env |
| `7` | Namespaces (только Linux) |

**Действия:**

| Клавиша | Действие |
|---------|----------|
| `K` / `Del` | Завершить процесс |
| `c` | Копировать строку |
| `p` | Копировать PID |
| `b` | Блокировать IP (firewall) |
| `t` | Подключить/отключить strace |
| `r` | Обновить |
| `s` | Sudo пароль |
| `Tab` | След. колонка сортировки |
| `Shift+Tab` | Направление сортировки |
| `L` | Переключить язык |
| `?` | Справка |

## Конфигурация

Создайте `~/.config/prt/config.toml`:

```toml
# Пользовательские имена портов
[known_ports]
3000 = "my-app"
9090 = "prometheus"

# Правила алертов
[[alerts]]
port = 22
action = "bell"

[[alerts]]
process = "python"
state = "LISTEN"
action = "highlight"
```

## Нас��ройка языка

Приоритет определения:

1. Флаг `--lang en|ru|zh` (высший приоритет)
2. Переменная окружения `PRT_LANG`
3. Автоопределение по системной локали
4. Английский (по умолчанию)

Клавиша `L` в TUI переключает язык без перезапуска.
